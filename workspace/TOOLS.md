# TOOLS.md — 本地筆記

Skills 定義工具*怎麼*運作。這個檔案是給*你的*具體細節——你的環境獨有的東西。

## 放什麼在這裡

像是：

- 攝影機名稱和位置
- SSH hosts 和 aliases
- TTS 偏好的聲音
- Speaker/房間名稱
- 設備暱稱
- 任何環境特定的東西

## 範例

```markdown
### 攝影機

- living-room → 主區域，180° 廣角
- front-door → 入口，動態觸發

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 偏好聲音："Nova"（溫暖，略帶英國腔）
- 預設 speaker：廚房 HomePod
```

## 為什麼分開？

Skills 是共享的。你的設定是你的。分開它們意味著你可以更新 skills 而不會丟失你的筆記，也可以分享 skills 而不會洩漏你的基礎設施。

---

加任何對你工作有幫助的東西。這是你的小抄。
