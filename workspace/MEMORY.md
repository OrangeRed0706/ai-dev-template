## 使用方式

- **只在主 session 載入**（跟你的人類直接聊天）

- 這是為了**安全**——包含不應該洩漏給陌生人的個人 context
- 你可以在主 session 自由**讀取、編輯、更新** MEMORY.md
- 寫重要事件、想法、決策、意見、學到的教訓
- 隨著時間，review 你的每日檔案並更新 MEMORY.md 保留值得留下的
